Driver Code
===========

This folder contains the driver code to be run on an STM32. Most of this folder is autogenerated by the IDE, and
so is a lot of the code. All the (non-generated) application code is under the folder `Code`.

Highlighted files:
.
├── Code
│   ├── Inc
│   │   ├── app_main.hpp # main application, inititialization and update code
│   │   ├── io.hpp # motor control, inputs and outputs
│   │   └── interrupts.hpp # interrupt handlers
│   └── Src # ...
├── Core
│   └── Src
│       ├── main.c # invokes the app_main functions, rest is autogenerated
│       ├── stm32f1xx_it.c # invokes most interrupts, also autogenerated
└── USB_DEVICE
    └── App
        └── usbd_cdc_if.c # invokes the USB send and receive functions, autogenerated


Tools needed:
* STM32 Cube IDE / STM32 Cube MX
